# Prototipo: La Velada del Año 6 - Experiencia Inmersiva

Este repositorio contiene un prototipo de alto rendimiento y diseño de vanguardia creado específicamente como propuesta para la interfaz web de **La Velada del Año 6**.

# Ver Demo en Vivo (Recomendado):
[https://prototipo-velada-vi-jorge-condado-c.vercel.app/](https://prototipo-velada-vi-jorge-condado-c.vercel.app/)

## Objetivo del Proyecto
El objetivo principal es evolucionar la experiencia digital del evento, pasando de una web estática a una **experiencia interactiva e inmersiva**. Cada sección ha sido diseñada para potenciar el *hype*, reflejando la magnitud de un evento de calibre mundial (estilo PPV de Boxeo / eSports de Tier 1).

## Destacados Técnicos
No es solo diseño, es rendimiento puro:
- **Animaciones a 60 FPS:** Uso avanzado de `framer-motion` con aceleración por hardware (GPU) (`will-change`, `transform-gpu`) para evitar caídas de frames durante el scroll.
- **Físicas Realistas:** Implementación de *spring physics* para el choque de los peleadores, dando sensación de peso e impacto.
- **Renderizado de Neón Optimizado:** Creación de efectos de neón multicapa usando `text-shadow` optimizados para no sobrecargar el hilo principal del navegador.
- **Stack Moderno:** React 18, Vite, Tailwind CSS v4 y Radix UI.
